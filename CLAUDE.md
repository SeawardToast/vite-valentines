# CLAUDE.md

## Project

Valentine's Day surprise website for my wife. Built with Vite + React + TypeScript.

## Key Documents

- [Project Overview](project_overview.md) — what we're building, design direction, color palette, animation goals
- [Coding Guidelines](coding_guidelines.md) — code style, rules, and conventions

## Quick Reference

- **Theme:** Romantic purple & pink, floaty, soft, lots of smooth animations and screen effects
- **Stack:** Vite, React, TypeScript, CSS
- **Code style:** Concise, simple, readable. No unnecessary abstraction. Short code > clever code.
- **Goal:** Make her smile and reveal a dinner reservation surprise

## Commands

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build
